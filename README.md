For CLI UI
- Run setup.sh
- Go to directory "logic" and run "pip install -e ." 
- Run "acktool" to test 

For Web UI
- python3 -m venv venv
- source venv/bin/activate
- Go to directory "FE" and run: npm install and npm start to have web UI
- Open other terminal, go to folder "logic" and run "python3 server.py"
